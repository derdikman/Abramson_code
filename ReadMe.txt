KrausData :  .mat files containing the data from each of the trials

NeuronFiringTimeVsDistance.m : matlab script analyzing a given trial 

RunAllSessionsTimeVsDistance.m : matlab script running the analysis of all trials 

Summary.mat : the output of all trials analysis

Onset analysis Dec-2021.xlsx : an excel file containing the data from summary.mat with further analysis

AllFinalResultsOnset+5sec_200mSec_bins.csv : csv file generated from the excel file with the analysis results

Summary.R : an R script using the above csv file and generating the analysis statistics

BarFigures.R : an R script using the above csv file and generating the paper bar figures
